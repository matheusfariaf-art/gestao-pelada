🔧 Problema Corrigido

Antes: � Sessões Participadas (emoji quebrado)

Depois: 📅 Sessões Participadas (emoji calendário)

📊 O que significa "Sessões Participadas"?

Definição: Quantidade de dias diferentes em que o jogador participou de pelo menos uma partida.

💡 Conceito: Uma "sessão" é um dia de pelada. Não importa quantos jogos o jogador fez naquele dia, conta como 1 sessão apenas.

📝 Exemplo Prático

📅
Segunda-feira (Sessão 1):
João jogou 3 partidas
📅
Quarta-feira (Sessão 2):
João jogou 2 partidas
📅
Sexta-feira (Sessão 3):
João jogou 1 partida

📊 Resultado Final para João:

  • ⚽ Partidas: 6 (total de jogos)
  • 📅 Sessões Participadas: 3 (3 dias diferentes)

🔍 Diferença Entre Estatísticas

Estatística O que conta
⚽ Partidas Cada jogo individual
📅 Sessões Cada dia que jogou (único)

🎯 Por que essa estatística é útil?

  • 📈 Frequência: Mostra a constância do jogador
  • 🗓️ Assiduidade: Indica quantos dias diferentes ele apareceu
  • 🔥 Dedicação: Diferencia quem joga muito num dia vs quem joga sempre
  • 📊 Comparação: Permite comparar frequência entre jogadores

⚙️ Como é calculado no código

JavaScript:

let sessoesParticipadas = new Set();
sessoesParticipadas.add(game.sessao_id);
// Set() evita duplicatas automaticamente

📝 Lógica: Para cada jogo que o jogador participou, adiciona o sessao_id em um Set. Como Set não permite duplicatas, conta apenas sessões únicas.
